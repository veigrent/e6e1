# vim-colors-e6e1

A Vim colorscheme with bg color #e6e6e1


## Install

Install with your favourite plugin manager. For vim-plug:

```viml
Plug 'veigrent/vim-colors-e6e1'
```

Then set your colorscheme to 'e6e1':

```viml
" in your .vimrc or init.vim
colorscheme e6e1
```
